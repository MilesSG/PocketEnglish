# 📱 PocketEnglish - 口袋英语

> 📚 让英语学习更轻松、更有效、更有趣！

## ✨ 功能特点

- 🔤 **单词学习** - 支持四级、六级和托福词汇，科学规划每日学习量
- 📝 **错词本** - 自动记录易错单词，针对性强化记忆
- 📅 **打卡记录** - 跟踪学习进度，养成良好学习习惯
- 💾 **本地存储** - 无需担心数据丢失

## 🛠️ 技术栈

- 基于Android原生开发
- Material Design设计风格
- SQLite本地数据库
- 支持Android 4.1及以上版本

## 🚀 开始使用

1. 下载并安装应用
2. 注册账号
3. 选择合适的词汇计划
4. 开始每日单词学习
5. 定期复习错词本

## 📊 项目架构

```
app/
├── src/main/
│   ├── java/
│   │   └── com/example/myapplication/
│   │       ├── Activity/       # 活动界面
│   │       ├── Bean/           # 数据模型
│   │       ├── SQL/            # 数据库操作
│   │       ├── Utils/          # 工具类
│   │       └── Calendar/       # 日历组件
│   └── res/
│       ├── layout/            # 布局文件
│       ├── values/            # 资源文件
│       ├── drawable/          # 图形资源
│       └── mipmap/            # 应用图标
```
